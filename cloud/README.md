# 云端服务目录说明

本目录是当前正式主线的云端服务实现目录。

## 当前定位
- 这是当前唯一权威的云端实现目录
- 当前阶段主要服务 A1 / A2 / A3 / A4 主线
- 旧版云端已归档为 `../cloud_ARCHIVED/`

## 当前职责
- 2D 地图展示与地图文件管理
- 机器人状态展示与任务状态广播
- 单点导航 / 多点导航 / 巡检任务下发
- 视频入口与前端实时展示
- 叶片温度历史预警相关数据承载
- 云端分析与建议生成的扩展入口

## 通信方式
- 与板端桥接通过 **WebSocket** 通信：`/ws/robot`
- 与前端通过 WebSocket 广播：`/ws`
- 视频推流使用 WebSocket：`/ws/video/push`、`/ws/video/watch`

## 重要边界
- 本目录不是“从零生成新系统”的实验目录
- 本目录是基于现有比赛主线持续迭代的正式实现
- **不要把早期 AI 草稿当成当前需求文档**
- 当前阶段中，LLM 是增强项，不是阻断主链路的核心依赖
- 当前 ESP32 方向是**叶片温度历史趋势预警**，不是普通温湿度看板项目
- 当前正式主线包含后续机械臂联动，因此云端设计不能忽略作业闭环
